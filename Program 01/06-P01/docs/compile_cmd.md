# Compile Command

```bash
g++ -std=c++17 -O2 -Wall -Wextra -pedantic -Iinclude src/main.cpp -o imgtool
```

# Breakdown

- g++ = C++ compiler
- -std=c++17 = use C++17
- -O2 = optimization
- -Wall = common warnings
- -Wextra = extra warnings
- -pedantic = strict C++ rules
- -Iinclude = look inside include folder
- src/main.cpp = source file
- -o imgtool = output executable
